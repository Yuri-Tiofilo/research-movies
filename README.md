# Search Movies

<!-- ##### Link de produção: https://marvel-search-heros-app.netlify.app/ -->

#### Ao clonar crie um ``.env`` semelhante ao ``.env.example`` 

## Como rodar o projeto:

### Instalar as dependencias
`````js
yarn install
`````

### Rodar a aplicação
`````js
yarn start
`````

### Commit

`````js
yarn commit
`````

### Run Build

`````js
yarn build
or
git push origin main // CI/CD Netlify
`````
